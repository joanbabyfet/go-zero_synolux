## About
使用 go-zero 微服务框架开发興溢公司官网前后端分离API接口

## Feature
* JWT身份验证
* 用户模块
* 文章模块
* 文件上传与缩略图功能
* 提供图片验证码功能
* 日志功能

## Usage
```
# 执行
go run user.go
go run article.go
go run feedback.go
```

## Requires
go v1.21.0  
goctl v1.6.5
